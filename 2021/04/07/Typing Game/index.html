JS

  • updateTime()
  • gameOver()
  1. setInterval()固定延遲了某段時間之後，才去執行對應的程式碼，然後「不斷循環」。
  2. 使用setInterval()才會開始倒數，少了這一行，time 變數就不會更動。
  3. 秒數歸 0 時，在 css 預先寫好的 style，讓他出現(display:flex)，預設為(display:none)。

jQuery

  • .style.display =轉成 .css("display", "flex")
