取得 天氣 API

先到中央氣象局註冊並且取得授權碼

  1. 註冊完成後，點選try it out，然後輸入Authorization。

  1. 因為下面還有許多欄位，一開始不知道還要輸入什麼，然後就在網路上看了其他人的文章也沒有人特別提及。所以，我就直接按下最下面的Execute，讓他產生連結，過程大約需要一分鐘。

  1. 產生 Request URL後就可以直接複製使用

  • console.log 來查看資料陣列

  • 官網有對照表，方便查找需要的資料

  • name 縣市的部分
  • Rain 降雨機率
  • WX 天氣狀況描述
  • MinT 最低溫
  • MaxT 最高溫
